# Flutter String Calculator - TDD Demo

A simple **String Calculator** built in **Flutter** following **Test-Driven Development (TDD)** principles.  
This project demonstrates addition of numbers from strings, supports custom delimiters, and handles edge cases like negative numbers.

---

## Key Features

- Addition of numbers from a string input  
- Supports **comma** and **newline** as default delimiters  
- Supports **custom delimiters** (single, multiple, or long delimiters)  
- Throws exception for **negative numbers** with proper messages  
- Fully covered with **unit tests**, each scenario in a separate test file  

---

## Technologies Used

- **Flutter** (Dart)  
- **Flutter Test** for unit testing  
- **Git** for version control  
- **GitHub Actions** for CI/CD (optional)
